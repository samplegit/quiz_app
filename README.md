# 간호조무사 국가시험 모의고사 웹 애플리케이션

간호조무사 국가시험 모의고사 문제를 웹 브라우저에서 풀 수 있는 퀴즈 애플리케이션입니다.

## 주요 기능

- **17개 회차 모의고사** (1-16회, 18회 / 17회는 PDF 파일 손상으로 제외)
- **105문제 × 5지선다형** 각 회차당
- **105분 타이머** 실제 시험과 동일한 시간 제한
- **1단/2단 레이아웃** 1문제 또는 2문제 동시 표시
- **과목별 성적 분석** 4개 과목(기초간호학, 보건간호학, 공중보건학, 실기)
- **합격/불합격 판정** 총점 60% 이상 + 각 과목 40% 이상
- **오답 리뷰 모드** 시험 후 정답/오답 확인

## 과목 구성

| 과목 | 문항 범위 | 문항 수 | 합격 기준 |
|------|----------|---------|----------|
| 기초간호학개요 | 1-35 | 35문항 | 40% 이상 |
| 보건간호학개요 | 36-50 | 15문항 | 40% 이상 |
| 공중보건학개론 | 51-70 | 20문항 | 40% 이상 |
| 실기 | 71-105 | 35문항 | 40% 이상 |

**최종 합격 기준**: 총점 60% 이상 + 모든 과목 40% 이상

## 사용 방법

### Windows

1. `start_server.bat` 더블클릭
2. 브라우저에서 `http://localhost:8080` 접속
3. 회차 선택 후 시험 시작

### Mac/Linux

```bash
cd quiz_app
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080 접속
```

## 주요 기능 설명

### 1. 시험 화면

- **타이머**: 105분 카운트다운, 남은 시간 10분 이하면 빨간색 경고
- **폰트 크기 조절**: 작게/보통/크게 3단계
- **레이아웃 전환**: 1단(1문제) / 2단(2문제) 버튼으로 전환
- **답안 패널**: 우측에 105개 문제 상태 표시 (미응답/응답됨)
- **과목별 탭**: 과목 클릭 시 해당 문제로 이동

### 2. 답안 선택

- 문제 카드의 선택지 클릭
- 답안 패널의 번호(1-5) 버튼 클릭
- **1단 모드에서만**: 키보드 1-5 숫자키로 선택 가능
- 같은 답 다시 클릭하면 선택 취소

### 3. 네비게이션

- **이전/다음 버튼**: 페이지 단위 이동 (1단=1문제, 2단=2문제)
- **방향키**: 좌우 화살표로 페이지 이동
- **답안 패널**: 문제 번호 클릭 시 해당 문제로 즉시 이동

### 4. 제출 및 결과

- **제출 버튼**: 미응답 문항 개수 표시 후 확인
- **자동 제출**: 시간 종료 시 자동으로 답안 제출
- **결과 화면**:
  - 과목별 점수 및 합격/불합격
  - 총점 및 최종 합격 여부
  - 정답/오답 표시
  - 정답 확인 가능

### 5. 리뷰 모드

- 정답은 **초록색** 표시
- 오답은 **빨간색** 표시
- 오답 선택 시 정답도 함께 표시
- 1단/2단 레이아웃 전환 가능
- 모든 문제 다시 확인 가능

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 반응형 디자인, Flexbox/Grid 레이아웃
- **Vanilla JavaScript** - 외부 라이브러리 없이 순수 자바스크립트
- **Python HTTP Server** - 로컬 개발 서버

## 데이터 소스

- **문제 PDF**: 간호조무사 국가시험 모의고사 1-18회 (17회 제외)
- **정답지**: 2개 이미지 파일에서 수동 전사
- **문제 텍스트**: PDF 이미지를 Claude AI로 OCR 추출

## 브라우저 호환성

- Chrome, Edge, Firefox, Safari 최신 버전
- 모바일 브라우저 지원 (반응형 디자인)

## 주의사항

- 17회차는 PDF 파일 손상으로 제외됨 (정답만 존재)
- 일부 문제는 이미지/도표가 포함되어 있어 `has_image: true` 플래그가 설정됨
- 로컬 서버 필수 (file:// 프로토콜로는 작동하지 않을 수 있음)
